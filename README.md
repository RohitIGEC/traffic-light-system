# Traffic Light System

## How it Works
- Red LED ON → 3 seconds → Stop
- Yellow LED ON → 1 second → Ready
- Green LED ON → 3 seconds → Go
- Repeats automatically

## Components
- Arduino Uno
- Red LED + Resistor
- Yellow LED + Resistor
- Green LED + Resistor

## Skills Used
- Embedded C programming
- Multiple output control
- Timing logic
